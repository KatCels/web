Como usar Git

Cómo usar Git 1. Inicialización del Repositorio Localmente Para iniciar un nuevo repositorio de Git, navega a la carpeta de tu proyecto y ejecuta: git init Inicialización de repositorio 2. Creación de Branches Para crear una nueva rama, utiliza el siguiente comando: git branch nombre-de-la-rama Para cambiar a la nueva rama: git checkout nombre-de-la-rama Creación de ramas 3. Agregar un Repositorio Remoto Para agregar un repositorio remoto, usa: git remote add origin url-del-repositorio
